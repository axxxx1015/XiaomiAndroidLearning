#### **作业目录结构**

```markdown
hello1/                      # 项目根目录
├── app/                     # 主模块
│   └── src/
│       ├── main/            # 主代码和资源
│       │   ├── java/com/example/hello1/  # Java代码包
│       │   ├── res/         # 资源文件
│       │   └── AndroidManifest.xml  # 应用配置清单
│       ├── androidTest/     # Android UI 测试代码
│       └── test/            # 本地单元测试代码
├── build.gradle             # 项目构建脚本
├── settings.gradle          # 项目配置
├── gradle/                  # Gradle Wrapper 配置
└── gradlew / gradlew.bat    

```
